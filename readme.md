# Reconstruction LLM

建築業者向け「建て替え vs リノベ」提案AIのPoCです。  
GPTを利用し、入力された物件情報から以下を出力します：

- 建て替え / リノベ / 解体の3案比較表
- 法令・制度上の留意点
- 提案文ドラフト
- 現地確認チェックリスト

## 使い方（予定）
1. `requirements.txt` を元に環境を構築
2. `src/prompt.py` のサンプルを実行
3. コンソールまたはノートブックで結果を確認
